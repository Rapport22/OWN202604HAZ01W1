OWN COACHING | WEEK 1

California Lifestyle

パターン（型）をマスターして、西海岸の魅力を語り尽くす

SET 1: THE CORE PATTERN

一番の「お気に入り」を強調する

The thing I love about
California is...
...the diverse culture and people.
...the relaxed atmosphere.
...the beautiful coastline.
SET 1: PRACTICE

例文でイメージを定着させる

The thing I love about California is the diverse culture and people.

カリフォルニアで大好きなのは、多様な文化と人々です。

The thing I love about California is the relaxed and laid-back atmosphere.

カリフォルニアで大好きなのは、リラックスしたのんびりとした雰囲気です。

SET 2: THE CORE PATTERN

「～だけでなく～も」と幅を広げる

It’s not only about
[A] but also about [B].
...tech / entertainment
...big cities / natural beauty
...high cost / high quality of life
SET 2: PRACTICE

対比を使って深みを出す

It’s not only about the tech industry but also about the entertainment world.

IT産業だけでなく、エンタメの世界も魅力です。

It’s not only about the big cities but also about the hidden natural beauty.

大都市だけでなく、隠れた自然の美しさも素晴らしいです。

SET 3: THE CORE PATTERN

「～すべきです」と自然に提案する

If you ever have a
chance, you should...
...drive along the Pacific Coast Highway.
...visit the local vineyards in Napa.
...experience an outdoor concert.
SET 3: PRACTICE

おすすめスポットを伝える

If you ever have a chance, you should drive along the Pacific Coast Highway.

機会があれば、パシフィック・コースト・ハイウェイをドライブすべきです。

If you ever have a chance, you should visit the local vineyards in Napa.

もし機会があれば、ナパの地元のワイナリーを訪れてみてください。

SET 4: THE CORE PATTERN

「どうなっているか」興味を語る

I’ve always been
interested in how...
...Silicon Valley changed the world.
...people balance work and life.
...the film industry started.
SET 4: PRACTICE

知的好奇心を表現する

I’ve always been interested in how Silicon Valley has changed the world.

シリコンバレーがいかに世界を変えたかにずっと興味がありました。

I’ve always been interested in how the film industry started in Hollywood.

ハリウッドでどのように映画産業が始まったかにずっと興味がありました。

REVIEW

Summary of Patterns
